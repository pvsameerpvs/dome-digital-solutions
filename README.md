# Dome Digital Solutions — Corporate Website

Modern, enterprise-grade company website built with:
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui-style components (locally included)

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Project Structure

- `app/` — Next.js App Router pages + layout + global styles
- `components/` — reusable UI components (shadcn-style)
- `sections/` — page sections (Hero, About, Solutions, Services, Brands, Contact)
- `lib/` — utilities and constants
- `public/` — logos and icons
